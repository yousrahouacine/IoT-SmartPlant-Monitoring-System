# IoT-SmartPlant-Monitoring-System
IoT-based Smart Plant monitoring system using ESP8266, MQTT, Flask backend, and Telegram integration.


# Smart Plant IoT System

This project implements an IoT-based Smart Plant monitoring system.

## Features
- Environmental monitoring (temperature, humidity, light, soil moisture)
- Rule-based plant status computation
- Weather-aware watering decision
- Telegram notification and user interaction
- Historical data storage
- LED remote control

## Architecture
ESP8266 → MQTT → Backend → Telegram / Web Interface

## Repository Structure
- /analysis → Project documentation
- /esp8266 → Microcontroller code
- /backend → Server-side API and logic
- /images → Architecture and wiring diagrams
